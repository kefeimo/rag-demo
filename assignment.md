# AI Engineer Coding Exercise

## Implement a Retrieval-Augmented Generation (RAG) System

---

## 🎯 Objective

The goal of this assignment is to evaluate your proficiency in using **retrieval-augmented generation (RAG)** techniques and your understanding of **vector databases**. You will be tasked with building a simple RAG system that retrieves relevant documents based on a user query and generates a response using a large language model (LLM).

---

## 📋 Tasks

### 1. Data Preparation
- **a.** Prepare a small dataset of text documents. These could be articles, code snippets, or any other text data you find suitable.

### 2. Backend Development
- **a.** Set up a **FastAPI** backend service.
- **b.** Implement a RAG system using an LLM and a vector database. The system should retrieve relevant documents based on a user query.
- **c.** Combine the retrieved documents with the prompt and generate a response using the LLM.

### 3. Evaluation
- **a.** Develop a simple evaluation framework to measure the quality of the generated responses.

### 4. Frontend Development (Optional - Big Plus)
- **a.** Build a user-friendly frontend interface using **React**.
- **b.** The interface should allow users to input their queries and display the generated responses.
